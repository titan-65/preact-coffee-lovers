# Coffee Lovers 


## Technical Requirements 

- Express
- React (Preact)
- MongoDB 
- Node.js

## Description

This application will be similar to instagram or any social media application in which person can take pictures of their favorite coffee.

### Features 

- All CRUD methods 
- Image upload 
- Input validation 
- Authentication and Authorization
- Maps 

## Node 

- Models 

User(s)

- firstName
- lastName
- email
- password 
- image
- dob
- address

Coffee Places

- title/name

- address
